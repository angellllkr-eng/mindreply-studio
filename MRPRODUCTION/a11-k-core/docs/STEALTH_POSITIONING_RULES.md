# Stealth Positioning Rules

## Purpose

Make all public-facing AUREL / A11-K / MindReply / Brushworks / tool surfaces feel premium, calm, trustworthy, and commercially understandable.

This is not deception.
This is controlled public positioning.

Do not expose internal architecture, private dashboards, credentials, model wiring, workflow providers, or implementation details.

## Brand hierarchy

1. AUREL = private execution layer
2. A11-K = quiet engine core / premium trust stamp
3. MindReply = public commercial offer/channel
4. Brushworks = visual build layer
5. Tools = simple product modules
6. Labs / ideas = experiments only

- protected command space

- workflow status
- action queue

- operating layer

- AI agent platform
- autonomous system
- shadow company
- model router
- backend dashboard

“AUREL is a private execution layer for decisions, workflows, actions, and protected execution.”

## Human wording replacements

Replace technical/internal words with public-safe wording:

- AI agent -> assistant / command support / guided execution
- model router -> provider selection / routing layer
- n8n -> workflow automation
- webhook -> workflow connection
- database -> secure workspace / data layer
- auth middleware -> private access gate
- Shadow Company -> Operating Twin
- DecisionTwin -> Decision Preview
- WorkflowTwin -> Workflow Status
- ModelTwin -> AI Model Status
- Autonomy Levels -> Action Safety Levels
- Escalation Rules -> When to Notify Angel
- rate limit -> usage limit
- provider key missing -> provider not connected yet
- Vercel limit hit -> deployment limit reached

## Privacy boundaries

Public pages must not show:
- secrets
- env values
- private logs
- customer data
- internal repo paths
- local machine paths
- raw provider names unless necessary
- n8n webhook URLs
- GitHub tokens
- Supabase service role keys
- private dashboard URLs
- owner-only workflows

## Status wording

Use honest status labels:

- Active
- Active locally
- Waiting for credentials
- Draft only
- Needs review
- Blocked
- Not verified

Never say live, active, automated, connected, or monitored unless true.

## CTA style

Use calm commercial CTAs:

- Open Command
- Start the Setup
- View the System Map
- Prepare a Safe Plan
- Check Workflow Status
- Review Next Action
- Request Private Setup
- Turn This Into a Workflow

Avoid hype CTAs:

- Launch autonomous AI
- Deploy agent army
- Replace your team
- Unlock secret system
- Run everything automatically
