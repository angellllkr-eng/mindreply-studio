# Compliance Guardrails

UNAPOLAGETIC starts with fashion/accessories/beauty tools.

Cosmetics are blocked until:
- supplier docs
- Responsible Person/importer path
- label fields
- ingredients
- batch/lot handling
- claims review
- country/market restrictions
- owner approval

No claims like cures, treats, medical result, guaranteed body change, unsafe beauty claim, counterfeit/replica claim.
